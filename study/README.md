## 一句话总结全书

```
提示词 → 告诉 AI 有什么工具和规则
AI Agent → 理解你的话，决定调什么工具
Skill → 执行具体的读写操作（30 个工具）
RAG → 搜索正文内容，找到相关段落
结果返回给 AI → AI 继续或回复你
```
