# 附录：ReWrite AI 系统代码结构一览

## 文件地图

所有与 AI Agent 相关的代码集中在一个目录下：

```
src/editor/modules/ai_assistant/          -- AI 助手模块根目录
│
├── agent.py                              -- AI Agent 核心
│   ├── AIAgent 类
│   │   ├── send_message()                -- 发消息给 AI
│   │   ├── undo_last_message()           -- 撤回上一条
│   │   ├── clear_history()               -- 清空记忆
│   │   └── _persist()                    -- 保存对话历史
│
├── providers.py                          -- API 供应商层
│   ├── get_proposals_only()              -- 第一轮：获取工具调用提案
│   ├── get_final_response()              -- 第二轮：获取最终回复
│   ├── _make_chat_request()              -- 通用 API 请求
│   ├── _describe_tool()                  -- 工具结果转文字
│   └── create_provider()                 -- 工厂：创建供应商实例
│
├── rag.py                                -- RAG 引擎（TF-IDF 搜索）
│   ├── RAGEngine 类
│   │   ├── build_index()                 -- 扫描章节，建立索引
│   │   ├── search()                      -- 搜索相关段落
│   │   └── search_formatted()            -- 返回格式化结果
│
├── contexts.py                           -- 上下文收集器
│   ├── collect_context()                 -- 主入口：收集所有模块的数据
│   ├── _characters_text()                -- 人物卡概览
│   ├── _outline_text()                   -- 大纲概览
│   ├── _timeline_text()                  -- 时间线概览
│   └── _map_text()                       -- 地图概览
│
├── prompt_templates.py                   -- 提示词模板
│   ├── SKILL_MAP                         -- 工具列表 + 规则
│   └── DEFAULT_SYSTEM_PROMPT             -- 默认系统提示词
│
├── markdown_render.py                    -- Markdown -> HTML 渲染
│   └── markdown_to_html()                -- 表格/加粗/斜体等渲染
│
├── module.py                             -- 模块入口，UI 绑定层
│   ├── _on_chat_message()                -- 用户发消息
│   ├── _on_tool_proposals()              -- AI 返回工具调用
│   ├── _execute_and_continue()           -- 执行工具后继续
│   ├── _tool_loop()                      -- 多轮递归循环
│   └── _refresh_panels()                 -- 刷新所有面板（含 RAG 重建）
│
├── worker.py                             -- 后台工作线程
│   ├── ToolProposalWorker                -- 第一轮：提案
│   ├── ToolExecuteWorker                 -- 第二轮：最终回复
│   └── ReasoningWorker                   -- 流式推理
│
├── annotation_manager.py                 -- 批注管理
│
├── skills/                               -- Skill 系统（30 个）
│   ├── __init__.py
│   ├── base_skill.py                     -- 基类
│   ├── registry.py                       -- 注册表
│   ├── _shared.py                        -- 共享工具函数
│   ├── character_skills.py               -- 人物卡（7 个）
│   ├── chapter_skills.py                 -- 章节（6 个）
│   ├── outline_skills.py                 -- 大纲（3 个）
│   ├── timeline_skills.py                -- 时间线（4 个）
│   ├── worldview_skills.py               -- 世界观（4 个）
│   ├── map_skills.py                     -- 地图（6 个）
│   └── rag_skills.py                     -- RAG 搜索（1 个）
│
└── ui/
    └── chat_panel.py                     -- AI 对话界面
```

## Skill 完整清单（共 30 个）

| 模块 | Skill 名称 | 功能 |
|------|-----------|------|
| 📖 章节 | `get_chapters` | 获取章节列表 |
| | `read_chapter` | 读取章节内容 |
| | `create_chapter` | 创建新章节 |
| | `update_chapter` | 修改章节（带 diff 确认） |
| | `rename_chapter` | 重命名章节 |
| | `delete_chapter` | 删除章节 |
| 👤 人物卡 | `get_character_groups` | 获取分组列表 |
| | `get_characters` | 获取角色数据（可分组过滤） |
| | `create_character` | 创建角色 |
| | `update_character` | 修改角色字段 |
| | `delete_character` | 删除角色 |
| | `add_group` | 添加分组 |
| | `delete_group` | 删除分组 |
| 📋 大纲 | `get_outline` | 获取大纲 |
| | `update_outline_entry` | 修改条目 |
| | `delete_outline_entry` | 删除条目 |
| 📅 时间线 | `create_timeline_event` | 创建事件 |
| | `get_timeline` | 获取时间线 |
| | `update_timeline_event` | 修改事件 |
| | `delete_timeline_event` | 删除事件 |
| 🌍 世界观 | `get_worldview` | 获取世界观 |
| | `create_worldview_entry` | 创建条目 |
| | `update_worldview_entry` | 修改条目 |
| | `delete_worldview_entry` | 删除条目 |
| 🗺️ 地图 | `get_map` | 获取地图数据 |
| | `create_map_node` | 创建节点 |
| | `update_map_node` | 修改节点 |
| | `delete_map_node` | 删除节点 |
| | `create_map_route` | 创建路线 |
| | `delete_map_route` | 删除路线 |
| 🔍 RAG 搜索 | `search_chapters` | 语义搜索章节正文 |

## 文件功能速查

| 你想做什么 | 看哪个文件 |
|-----------|-----------|
| 加一个新 Skill | `skills/xxx_skills.py` + `skills/registry.py` + `prompt_templates.py` |
| 改 AI 的行为规则 | `prompt_templates.py` 中的 SKILL_MAP |
| 改 AI 能看到哪些上下文 | `contexts.py` 中的 collect_context |
| 改记忆持久化逻辑 | `agent.py` 中的 _persist / load 方法 |
| 改 RAG 搜索逻辑 | `rag.py` 中的 RAGEngine |
| 改确认气泡 | `module.py` 中的 _on_tool_proposals |
